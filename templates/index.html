CS 531 WebServer Deployer and Tester

A deployment and testing system for the CS 531 (Web Server Design) course projects. The system utilizes a Docker-based setup for both, this testing/deployment system as well as deploying students' projects. The code behind this system is available at github.com/ibnesayeed/webserver-tester. Please feel free to report issues and contribute.
